算术运算符

运算符 描述 实例
+ 两个数相加 2+3.9 结果为 5.9
- 两个数相减 -2-3.9 结果为 -5.9
* 两个数相乘 2*3.9 结果为 7.8
/ 两个数相除 3.9/2 结果为 1.95
% 模运算得余数 3.9%2 结果为 1.9
** 乘幂运算 2**3.9 结果为 15.2099999
// 取商的整数部分(向下取整) -9//2 结果为 -5

身份运算符

身份运算符用于比较两个对象的地址
id()函数用于获取对象的地址,类似C语言中的&

运算符 描述
is 判断两个标识符是不是引用自一个对象
is not 判断两个标识符是不是引用自不同的对象 
    a=20					
    b=20					
    print(a is b)			#True
    print(a is not b)		#False

    #修改变量b的值
    b=30
    print(a is b)			#False
    print(a is not b)		#True

is与==的区别:

        a = [1, 2, 3]
        b = a
        b is a				#True
        b == a				#True
        b = a[:]
        b is a				#False
        b == a				#True

条件语句

Python不支持switch语句

    if 判断条件1:
        执行语句1......
    elif 判断条件2:
        执行语句2......
    else:
        执行语句3......

循环语句

while循环

while 判断条件:
    执行语句1
else:
    执行语句2		#不满足判断条件时执行一次

通过break语句跳出循环不会进入else语句
使用CTRL+C跳出无限循环

for循环

    for letter in 'python':
        print("当前字母 : %s" % letter)
    fruits = ['apple', 'banana', 'orange']
    for index in range(len(fruits)):
        print('当前水果 : %s' % fruits[index])

循环嵌套

    #输出小于100的所有素数
    for num in range(101)[2:]:
           for i in range(int(num ** 0.5) + 1)[2:]:
            if num % i == 0: break
        else: print(num)
