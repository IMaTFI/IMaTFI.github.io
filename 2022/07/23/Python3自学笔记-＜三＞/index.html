Python3自学笔记 ＜三＞

2022-07-23

数字

参考菜鸟教程https://www.runoob.com/python/python-tutorial.html

数字类型转换

  • int(x)将x转换为一个整数
  • float(x)将x转换为一个浮点数
  • complex(x)将x转换为一个复数, 实部为x, 虚部为0
  • complex(x， y)将x转换为一个复数, 实部为x, 虚部为y

数学函数

*代表需要import math

函数 输入 功能
abs(x) 实数, 复数 返回实数的绝对值或复数的模
* ceil(x) 实数 向上取整
* exp(x) 实数 计算e的x次方
* floor(x) 实数 向下取整
* log(x,y) 实数 以y为底,x的对数
max() 实数序列 返回序列中的最大值(通过运算符>实现)
min() 实数序列 返回序列中的最小值
* modf() 实数 返回(x,y), x为参数的小数部分,y为参数的整数部分
round(x,y) 实数 对x四舍五入,保留小数点后y位
* sqrt(x) 实数 返回x的算数平方根

字符串

f-string

f-string格式化字符串以f开头,后面紧跟字符串,字符串中的表达式用{}抱起来,它会将变量或表达式计算后的值替换进去

str = 'world'
print(f'Hello,{str}')			#Hello,world

在Python3.8版本后,可以使用=拼接运算表达式与结果

a = 1
print(f'{a+1}')		#2
print(f'{a+1=}')	#a+1=2

常用方法

调用方式: 字符串.方法()

方法 功能
find(str, beg=0, end=len(string)) 检测str是否包含在字符串中, beg和end用来指定范围, 如果包含则返回开始的索引值,否则返回-1
rfind(str, beg=0, end=len(string) 与find()功能一致,实现方式是从右查找
join(seq) 将序列中的元素以指定的字符连接生成一个新的字符串 
s1 = '-'
seq = ('h', 'e', 'l', 'l', 'o')
print(s1.join(seq))			#h-e-l-l-o
函数 功能
replace(old, new [,max]) 将字符串中的old替换成new. 如果max指定, 则替换不超过max次

列表

常用函数

函数 功能
len() 列表元素个数
max() 返回列表中元素最大值
min() 返回列表中元素最小值
list(seq) 将元组seq转换为列表

常用方法

方法 功能
append(obj) 在列表末尾添加元素
count(obj) 统计某个元素在列表中出现的次数
extend(seq) 在列表末尾一次性添加多个值
index(obj) 从列表中找出指定值的第一个索引位置, 没找到则抛出异常
insert(index,obj) 将元素插入列表中指定的索引位置
pop(index=-1) 类似出栈操作
remove(obj) 移除某个值的第一个匹配项, 没有返回值
reverse() 翻转列表
clear() 清空列表
copy() 复制列表
sort(key=None, reverse=False) 对原列表进行排序(默认升序) 
L1=['h', 'e', 'l', 'l', 'o']
L1.sort()					#['e', 'h', 'l', 'l', 'o']
L1.sort(reverse=True)		#['o', 'l', 'l', 'h', 'e']

L2=[(1, 2), (2, 1), (3, 9), (4, 0)]
def takeSecond(elem):
    return elem[1]
L2.sort(key=takeSecond)		#[(4, 0), (2, 1), (1, 2), (3, 9)]
