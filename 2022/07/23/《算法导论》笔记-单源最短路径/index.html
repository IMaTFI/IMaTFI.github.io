《算法导论》笔记__单源最短路径

2022-07-23

符号说明

  • $V$：结点集；
  • $E$：边集；
  • $G=(V,E)$：带权重的有向图；
  • $w$：权重函数；
  • $\delta(u,v)$：结点$u$到结点$v$的最短距离。

最短路问题的最优子结构

运用“剪切-粘贴”技术。已知$p=\langle v_0,v_1,…,v_k\rangle$为结点$u$到结点$v$的最短路径。假设$p$中的$\langle v_i,…,v_j\rangle$不是结点$i$到结点$j$的最短路径，则一定有：$\delta(v_0,v_i)+\delta(v_i,v_j)+\delta(v_j,v_k)<w(p)$，这与$p$是结点$v_0$到结点$v_j$的最短路径相矛盾！

证明了最短路问题的最优子结构，我们便可以尝试利用动态规划和贪心算法求解问题。

松弛操作

对于每一个结点$v$，我们维持属性$v.d、v.\pi$，分别表示表示源点$s$到结点$v$的最短距离的上界，和结点$v$的前驱节点。
对边$(u,v)$进行松弛操作的过程为：

RELAX(u, v, w)
    if v.d > u.d + w(u, v)
        v.d = u.d + w(u, v)
        v.pi = u

初始化

INITIALIZE-SINGEL-SOURSE(G,s)
    for each vertex v in G.V
        v.d = inf
        v.pi = -1
    s.d = 0

最短路径和松弛操作的性质

性质1：三角不等式性质

内容：

对所有的边$(u,v)$，有

证明

假设$p$为源点$s$到结点$v$的最短路，则$p$的权重一定小于等于任意一条源点$s$到结点$v$的路径的权重。也就是说，$p$的权重一定小于等于$s$到$u$的最短路，再加上边$(u,v)$的路径权值。对于$s,v$之间不存在最短路的情况，易证$s,u$之间也不存在最短路，此时有$\infty\leq\infty+w(u,v)$，原不等时仍成立。

性质2：上界性质

内容

对图$G$执行初始化操作INITIALIZE-SINGEL-SOURSE(G,s)后。在松弛过程中，对于所有结点$v$，$v.d\ge\delta(s,v)$恒成立。而且，一旦$v.d$取其下界$\delta(s,v)$后，取值将不再发生变化。

证明

采用数学归纳法证明不变式

  • 在初始化操作结束后，$v.d\geq\delta(s,v)$显然成立。
  • 假设在对边$(u,v)$执行松弛操作前结论成立，在对边$(u,v)$执行松弛操作后，唯一可能改变的值只有$v.d$。假设$v.d$发生变化，则有此时不变式仍然成立。

由于$v.d$在松弛过程中只能减小，不能增加，而上文也已经证明$v.d\geq\delta(s,v)$，所以$v.d$取到下界后将不再发生变化。

性质3：非路径性质

内容

假设$s,v$之间不存在路径，在松弛过程中始终有$v.d=\delta(s,v)=\infty$。

证明

根据上界性质，始终有$v.d\ge\delta(s,v)=\infty$，又由于$v.d$的初值为$\infty$，所以性质成立。

性质4：路径松弛性质

内容

考虑从源点到的任意一条最短路径，如果对图G进行初始化，并按照次序对边进行松弛后，有。

证明

采用数学归纳法，归纳假设为在对路径p中的第$i$条边进行松弛操作后，有$v_i.d=\delta(s,v_i)$

  • $i=0$时，显然有$v_0.d=0=\delta(s,s)$。
  • 假设对第$i-1$条边进行松弛操作后有，在对第条边进行松弛操作后，显然有成立。

Bellman-Ford算法

Bellman-Ford可以处理存在负权边的最短路问题。

BELLMAN-FORD(G,w,s)
    INITIALIZE-SINGLE-SOURCE(G,s)
    for i = 1 to |G.V|-1
        for each (u,v) in G.E
            RELAX(u,v,w)
    for each (u,v) in G.E
        if v.d > u.d+w(u,v)
            return FALSE
    return TRUE

直白地说，Bellman-Ford算法就是对图中的每条边执行$|V|-1$次松弛操作，时间复杂度为$O(VE)$。

Bellman-Ford算法的正确性

首先证明，当图中不包含负权环时，执行Bellman-Ford算法后，对所有点有$v.d=\delta(s,v)$且返回值为TRUE。由于源点$s$到$v$的最短路径一定为简单路，所以任意一条最短路径p中包含的边数$k\le |V|-1$。在执行第$i$轮松弛操作的过程中，一定对边$(v_{i-1},v_i)$进行了松弛，根据路径松弛性质，$v.d=\delta(s,v)$。特别地，当$s,v$之间不存在路径时，$v.d$自始至终没有被修改，有$v.d=\infty$。根据三角不等式$\delta(s,v)\le\delta(s,u)+w(u,v)$ ，证明不会有$v.d>u.d+w(u,v)$成立，所以算法最后的返回值一定为TRUE。

Dijkstra算法

Dijkstra算法要求所有边的权值非负。

DIJKSTRA(G,w,s)
    INITIALIZE-SINGLE-SOURCE(G,s)
    s = 空集
    Q = G.V
    while Q != 空集
        u = EXTRACT-MIN(Q)
        S=S∪{u}
        for each vertex v in G.Adj[u]
            RELAX(u,v,w)

直白地说，Dijkstra算法在执行过程中维持一个点集$S$，用以保存最短路径已经被找到的点。算法重复从$V-S$中找到$d$值最小的点$u$，将其加入集合$S$中，然后对所有$u$出发的边进行松弛。

Dijkstra算法的正确性

证明Dijkstra的正确性，等价于证明这个循环不变式：在算法的执行过程中，每当将新的$u$加入$S$时，有$u.d=\delta(s,u)$。

  • 初始时，$S=\varnothing$，循环不变时成立。
  • 采用反证法，假设结点$u$为第一个不满足循环不变式的结点，即$u.d\ne\delta(s,u)$。我们关注将结点$u$加入$S$的那一轮循环。首先$u\ne s$，因为当$s$加入集合$S$时，有$s.d=0=\delta(s,s)$。再者，$s,u$之间一定存在最短路，因为如果$s,u$之间不存在最短路，则有$u.d=\infty=\delta(s,u)$。我们假设源点$s$到结点$u$的最短路径为$p$，设这条路径上第一个$\notin S$的结点为$y$，设$y$的前驱节点为$x$。此时，最短路径$p$可以表示为$s\leadsto x\to y\leadsto u$。因为$x\in S$，所以$x.d=\delta(s,x)$，且当$x$加入$S$时，边$(x,y)$已经被松驰过了。所以有$y.d=\delta(s,y)$。因为在路径$p$上，$y$在$u$的前面，所以$y.d=\delta(s,y)\leq\delta(s,u)\leq u.d$。但是，我们注意到，当轮循环中，加入$S$的是$u$，所以一定有$u.d\leq y.d$。所以$y.d=\delta(s,y)=\delta(s,u)= u.d$，这与$u.d\ne \delta(s,u)$相矛盾！

Dijkstra算法的时间复杂度

Dijkstra算法的时间复杂度主要取决于最小优先队列的实现方式。

  • 数组：插入、修改结点的时间复杂度均为$O(1)$，找到最小元素的时间复杂度为$O(V)$，总的时间复杂度为$O(V^2)$
  • 二叉堆：插入、修改结点的时间复杂度为$O(logV)$，找到最小元素的时间复杂度为$O(logV)$，总的时间复杂度为$O((V+E)logV)$。适用于稀疏图。

差分约束

如果一个系统由$n$个变量和$m$个约束条件组成，形成$m$个形如$x_j-x_i ≤ b_k$的不等式($i,j∈[1,n]$,$b_k$为常数),则称其为差分约束系统(system of difference constraints)。

首先引入一个简单的引理：若向量$x=(x_1,x_2,…,x_n)$为差分约束系统的一个解，则$x+d=(x_1+d,x_2+d,…,x_n+d)$也是差分约束系统的一个解。

约束图

给定一个差分约束系统$Ax\leq b$，其对应的约束图是一个带权有向图$G(V,E)$：

  • $V=\{v_0,v_1,…,v_n\}$
  • $E=\{(v_i,v_j):x_j-x_i\leq b_k\}\cup \{(v_0,v_1),(v_0,v_2),…,(v_0,v_n)\}$
  • 如果$x_j-x_i\leq b_k$，则$w(i,J)=b_k$，特别地，$w(v_0,v_i)=0$.
