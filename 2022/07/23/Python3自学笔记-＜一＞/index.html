Python3基础语法

参考菜鸟教程 https://www.runoob.com/python/python-tutorial.html

注释

  • 单行注释:以#开头
  • 多行注释:'''或"""

    多行语法

  • 使用反斜杠\实现多行语句

          str = "hello" + \
              "world"

  • 在[],{},()的多行语句中,不需要使用\

          str = ["hello" , 
              "world"]

字符串

  • 单行字符串用'或"指定
  • 多行字符串用'''或"""指定

  • 反斜杠\可以用来转义,使用r可以让反斜杠不发生转义

      print(r"hello\nworld")            #输出hello\nworld

  • python的字符串有两种索引方式,从左向右以0开始,从右向左以-1开始

  • 字符串截取:变量[头下标:尾下表:步长](尾下表的字符不输出)

      str = "123456"
  print(str)                  #123456
  print(str[0:-1])          #12345
  print(str[3:])              #45
  print(str[0::2])            #135
  print(str[0::-1])            #1

print输出

  • print输出默认换行,如果不需要换行,则要在变量末尾加上end=""

      print("a", end="")
  print("b", end="")
  #输出:ab

Python3基本数据类型

python中的变量没有类型,每个变量在使用前都必须被赋值,变量被赋值以后该变量才会被创建

多变量赋值

  • 将多个变量赋予相同的数值

      a, b, c = 1

  • 为多个变量赋予不同的值

      a, b, c = 1, 1.0, "hello"

标准数据类型

  • 不可变: Number, String, Tuple
  • 可变: List, Dictionary, Set
  • 内置函数type()函数可以用来查询变量类型
  • 内置函数isinstance()函数可以用来判断变量类型

  • type()和ininstance()的区别在于:
    type()不会认为子类是一种父类类型,而isinstance()会

      class A:
      pass

  class B(A):
      pass

  type(A()) == A                #True
  isinstance(A(), A)            #True

  type(B()) == A                #False
  isinstance(B(), A)            #True

    Number(数字)

    Python支持int, float, bool, complex

  • 数值运算

      5 + 4                    #9        加法
  3.5 - 1                #2.5    减法
  2 * 5                    #10        乘法
  2 / 4                    #0.5    除法, 结果为浮点数
  2 // 4                #0        除法,结果为整数
  15 % 2                #1        模运算
  2 ** 5                #32        乘方

String(字符串)

见上文

List(列表)

  • 列表中元素的类型可以不同,支持数字,字符串,甚至列表
  • 列表用方括号[]定义,列表内的元素用逗号,分隔
  • 列表截取同字符串相同,列表被截取后返回一个包含所需元素的新列表
  • 列表支持+和*运算

    Tuple(元组)

  • 元组与列表类似,不同之处在于元组的元素不能修改
  • 元组用()定义
  • 可以把字符串看作特殊的元组

  • 包含0个或1个元素的元组比较特殊,有额外的语法规则

      tup1 = ()        #空元组
  tup2 = (20,)    #一个元素,需要在元素后面添加逗号

    Set(集合)

  • 与数学中的集合定义类似
  • 集合中不允许有可变元素类型?

  • 可以使用大括号{}或者set()函数构造集合.创建一个空集必须使用set()

      set1={'a', 1 , 3.14}
  set2=set('helloworld')        #set2={'l', 'h', 'd', 'e', 'w', 'o', 'r'}

  • 集合运算

      A = set("abcdef")
  B = set("defghi")
  print(A - B)            #差集
  print(A | B)            #并集
  print(A & B)            #交集
  print(A ^ B)            #并集-交集
  'a' in A                #True

    Dictionary(字典)

  • 字典是一种映射类型,用{}标识,是无序的键(key):值(value)的集合

  • 在同一个字典中,键必须是唯一的,且必须为不可变类型

      dict = {}
  dict["one"] = "a"
  dict[2] = "b"
