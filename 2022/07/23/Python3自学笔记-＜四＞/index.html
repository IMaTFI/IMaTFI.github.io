迭代器

参考菜鸟教程https://www.runoob.com/python3/python3-tutorial.html

L=[1, 5, 8, 3, 6, 7]
it = iter(L)            # 创建迭代器对象
print(next(it))            # 1
print(next(it))            # 2
for x in it:
    print(x, end=" ")    # 8 3 6 7

生成器

使用了yield的函数成为生成器, 其返回值是一个迭代器对象

def fib(n):
    a, b = 0, 1
    while b<n:
        yield b
        a, b = b, a+b
f = fib(20)                    # f是一个迭代器
for x in f:
    print(x, end=" ")        # 1 1 2 3 5 8 13

参数传递

在python中, 变量是没有类型的, 它仅仅是一个对象的引用(指针), 可以指向任何对象

  • 参数为不可变类型: 类似C++中的值传递

      def change(a):
      print(id(a))        # 1443155044656
      a = 10
      print(id(a))        # 1443155044654, 生成了一个新的对象
      return

  a = 1
  print(id(a))            # 1443155044656
  change(a)
  print(id(a))            # 1443155044656, a并没有受到函数的影响

  • 参数为可变类型: 类似C++中的引用传递

      def change(L):
      L.append(1)
      return

  L=[1, 2, 3, 4]
  print(L)                    # [1, 2, 3, 4]
  change(L)
  print(L)                    # [1, 2, 3, 4, 1]

  • 带星号*的参数会以元组的形式导入

      def printinfo(arg1, *varTuple):
  print(arg1)
  for x in varTuple:
      print(x)
  return

  printinfo(10)
  printinfo(10, 20, 30, 40)
  • 带两个星号*的参数会以字典的形式导入
