自底向上计算最短路径权重

根据邻接矩阵$W$，我们可以计算出矩阵序列$L^{(1)},L^{(2)},\cdots,L^{(n-1)}$。特别地，$L^{(1)}=W$。

下面的伪代码展示了如何根据$W$和$L^{(m-1)}$计算$L^{(m)}$：

EXTEND-SHORTEST-PATHS(L,W)
    n = L.rows
    let L' be a new n*n matrix
    for i = 1 to n
        for j = 1 to n
        L'[i][j]=∞
        for k = 1 to n
            L'[i][j] = min(L'[i][j], L[i][k]+w(k,j))
    return L'

由于上述算法有3层嵌套for循环，所以时间复杂度为$O(n^3)$。

可以看出，EXTEND-SHORTEST-PATHS算法和矩阵乘法在形式上是非常相似的。

下面的算法给出了如何在$O(n^4)$的时间内计算所有节点对的最短路径：

SLOW-ALL-PAIRS-SHORTEST-PATHS(W)
    n = W.rows
    L(1) = W
    for m = 2 to n-1
        let L(m) be a new n*n matrix
        L(m) = EXTEND-SHORTEST-PATHS(L(m-1),W)
    return L(n-1)

改进算法的运行时间

我们知道：$\forall m>n-1\Longrightarrow L^{(m)}=L^{(n-1)}$。所以我们在求解所有结点对的最短路径时，未必一定要计算出$L^{(n-1)}$。
下面的算法采用重复平方技术，使算法的时间复杂度降至$O(n^3\log n)$：

FASETER-ALL-PAIRS-SHORTEST-PATHS(W)
    n = W.rows
    L(1) = W
    m = 1
    while m < n-1
        let L(2*m) be a new n*n matrix
        L(2*m) = EXTEND-SHORTEST-PATHS(L(m),L(m))
        m = 2*m
    return L(m)

Floyd-Warshall算法

最短路径的结构

Floyd-Warshall算法主要关注最短路径上的中间结点。

假定图$G$中的所有结点为${1,2,\cdots,n}$，考虑其中的一个子集${1,2,\cdots,k}$，对于任意的结点对$i,j\in V$，设$p$为所有中间结点均取自集合${1,2,\cdots,k}$的路径中的最短路。对于路径$p$，我们讨论以下两种情况：

  • $k$不在路径$p$上：p也是中间结点均取自集合${1,2,\cdots,k-1}$的路径中的最短路。
  • $k$在路径$p$上：将p拆分为：$i\leadsto^{p_1}k\leadsto^{p_2}j$，$p_1$为结点$i$到$k$的、所有中间结点取自集合${1,2,\cdots,k-1}$的最短路径，$p_2$同理。

    所有结点对最短路径问题的递归解

    设$d^{(k)}{ij}$为从结点$i$到结点$j$的、所有中间结点均取自集合${1,2,\cdots,k}$的最短路径权值。$d^{(0)}{ij}$表示$i,j$的路径上没有任何中间结点，所以有：$d^{(0)}{ij}=w{ij}$。递归定义$d^{(k)}_{ij}$如下：因为对于图$G$中的任何路径来说，所有中间结点均取自集合${1,2,\cdots,n}$，所以$d^{(n)}_{ij}=\delta(i,j)$。

    自底向上计算最短路径权重

    FLOYD-WARSHALL(W)
      n = w.rows
  D(0) = W
  for k = 1 to n
      let D(k) be a new n*n matrix
      for i = 1 to n
          for j = 1 to n
              d(k)[i][j] = min(d(k-1)[i][j], d(k-1)[i][k]+d(k-1)[k][j])
  return D(n)
    时间复杂度为$O(n^3)$。

Johnson算法

使用斐波那契堆实现最小优先队列时，Johnson算法的时间复杂度为$O(V^2\log V+VE)$。使用二叉堆实现最小优先队列时，时间复杂度为$O(VE\log V)$在算法的执行过程中需要使用Dijkstra和Bellman-Ford算法。

Johnson算法使用了重赋权重技术。具体如下：

  • 如果图中均为非负权值的边，则对每个结点执行Dijkstra算法，从而找到所有结点对的最短路径。
  • 如果图中包含负权边，但不包含负权环，则需要计算出一组新的权重，然后再对每个结点调用Dijkstra算法。

新的权重$\hat{w}$必须满足如下性质：

  • 对任意结点对$u,v$，如果在原权重$w$下，存在一条最短路径$p$当且仅当在新权重下$\hat{w}$下，$p$仍然为两结点间的最短路径。
  • 对于所有的边$(u,v)$，新权重$\hat{w}(u,v)$为非负值。

    重赋权重来维持最短路径

    令$\hat{w}(u,v)=w(u,v)+h(u)-h(v)$，我们假设在新权重$\hat{w}$下的最短路径权重为$\hat{\delta}$。

首先证明，对$v_0,v_k$的最短路$p=\langle v_0,v_1,\cdots,v_k\rangle$，$w(p)=\delta(u,v)$当且仅当$\hat{w}(p)=\hat{\delta}(u,v)$。对任意路径，我们有:

由于$h(v_0),h(v_k)$不依赖于任何具体路径，所以新权重下的路径之间的大小关系保持不变，由此证明了：$w(p)=\delta(u,v)$当且仅当$\hat{w}(p)=\hat{\delta}(u,v)$。

然后证明，若原权重$w$下存在负权环，则新权重$\hat{w}$下仍然存在负权环。考虑任意环路$c=\langle v_0,v_1,\cdots,v_k\rangle,v_0=v_k$，我们有$\hat{w}(c)=w(c)+h(v_0)-h(v_k)=w(c)$。命题得证。

通过重赋权值来生成非负权重

我们制作一幅新图$G’=(V’,E’)$，其中$V’=V\cup{s},E’=E’\cup { (s,v):v\in V}，w(s,v)=0$。
我们定义：对所有结点$v\in V’$，$h(v)=\delta(s,v)$。根据三角不等式我们知道：$\delta(s,v)\leq\delta(s,u)+w(w,v)$，变形得：$\delta(s,u)-\delta(s,v)\geq w(w,v)$，即$h(u)-h(v)\ge w(u,v)$，从而有$\hat{w}(u,v)=w(u,v)+h(u)-h(v)\ge 0$。
