算法自学__堆优化的迪杰斯特拉算法

2022-07-23

需要用到的STL知识

pair

#include<utility>
pair <int, int> p;		//创建
p.first = 1, p.second = 2;		//访问pair中元素, 赋值

priority_queue

#include<queue>
typedef pair<int, int> Pair;		//first代表边的终点, second代表路径长度
class cmp {			//仿函数
public:
    bool operator()(Pair a, Pair b) {
        return a.second > b.second;		//小根堆
    }
};
vector<Pair> graph[10005];		//创建

迪杰斯特拉算法

  • 功能: 求解有向图单源最短路问题(图中不允许出现负权值)
  • 算法流程:
    1. 设起点为s, 初始化结果数组ans[],将ans[s]赋值为0, 其余元素赋值为无穷大
    2. 将起点s能到达的点对应的ans[]中的元素更新为边的权值
    3. 选择最小权值的边的终点重复步骤2.

堆优化

利用优先队列保存算法执行过程中的边的权值, 将选择最小权值的边的时间复杂度降低到了O(nlogn)

void dijkstra(int x){
    for(int i=1;i<=n;i++){
        ans[i]=2147483647;		//int的最大值 
    }
    ans[x]=0;		//起点 
    q.push(make_pair(x, 0));		//起点入队 
    while(!q.empty()){
        Pair p=q.top();
        q.pop();
        int pos=p.first;
        if(vis[pos]!=0) continue;		//对出队的点进行访问 
        vis[pos]=1;
        for(int i=0;i<graph[pos].size();i++){		
            int num=graph[pos][i].first;
            int dis=graph[pos][i].second;
            ans[num]=min(ans[num],ans[pos]+dis);
            q.push(make_pair(num,ans[num]));
        }
    }
}
