算法简介

筛法的基本原理是：将素数的所有倍数（除了该素数本身）全部筛除。

线性筛（欧拉筛）能够在线性的时间复杂度内筛选出一定范围内的素数。实现该算法的关键在于：保证每个合数仅被筛除一次。具体来说，就是让一个合数仅被其最小的质因子筛除，如，合数30仅被2筛除。

代码

if (i%prime[j] == 0) break;保证了合数永远只会被它最小的质因子筛除。还是以为例，由于if (i%prime[j] == 0) break;的存在，保证了只会在i==15 && prime[j]==2时被筛除。为什么不会在i==10 && prime[j]==3时被筛除呢？因为在遍历prime[]时，显然2在3之前被访问到，这时就会有10%2 == 0。
